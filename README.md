# Rat PDF Tools

Rat PDF offers free online PDF tools to **merge, split, compress, convert, and edit PDFs**.  
No downloads or installations required — just your browser.

## Features

- Merge PDFs  
- Split PDFs  
- Compress PDFs  
- Convert PDF ↔ Word / Excel / Images  
- Add Watermarks & Password Protection  

Try it online: [https://ratpdf.com](https://ratpdf.com)
